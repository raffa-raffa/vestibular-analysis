# 🎓 Vestibular Data Analysis

Análise exploratória de dados de um vestibular real, utilizando **Python**, **Pandas** e **GoogleCollabs** e **VS Code**.

## 🔍 Objetivo
Identificar insights sobre notas e desempenho, como a maior nota no curso de Computação e o polo correspondente.

## ⚙️ Tecnologias
- Python 3
- Pandas
- OpenPyXL
- Google Collab
- VS Code

## 🧩 Etapas
1. Leitura do arquivo Excel (`.xlsx`)
2. Limpeza e padronização das colunas
3. Filtragem de dados do curso de Computação
4. Cálculo da maior nota e exibição do aluno e polo

## 🚀 Como rodar
1. Clone este repositório  
2. Instale as dependências:
   ```bash
   pip install pandas openpyxl
3. Execute 
   ```bash
   python src/main.py
